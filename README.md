
## 简介

根据鸿洋大神的WanAndroid开放api开发的一个简单的flutter版本的WanAndroid客户端

## 功能

* 登录
* 注册
* 首页banner和文章列表
* 体系
* 导航
* 项目
* 收藏
* TODO（便签）
* 搜索

## 涉及技术点

``` 
dio: 2.1.7  //网络请求库
toast: ^0.1.5  //Toast
flutter_swiper : ^1.1.6  //滚动页面，类似ViewPager
shared_preferences: ^0.5.3+4  //sp存储
flutter_webview_plugin: ^0.3.5  //webview
date_format: ^1.0.6  //日期格式化
```
